# App Language Converter

A Node.js application that converts code from one programming language to another using Google's Gemini AI. It clones a GitHub repository, identifies code files, and converts them to the specified target language.

### How to setup application ?

```sh
1. Create an API Key using url. (https://aistudio.google.com/app/apikey)
2. Create .env file in the application and add Gemini AI API Key like this:
GEMINI_AI_API_KEY=your api key
3. Execute command `npm install` in application root
4. Install nodemon package in your machine using command `npm install -g nodemon`
5. Execute command `nodemon start`
```
